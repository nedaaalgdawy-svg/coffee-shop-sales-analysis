# ☕ Coffee Shop Sales Analytics

## Project Overview
Analysis of 3,896 coffee shop transactions (March 2024 – March 2025)
Total Revenue: 122,272 | Avg Order Value: ~31.4

## Key Insights
- Latte & Americano With Milk = 45% of total revenue
- 417% YoY growth in Q1 2025 vs Q1 2024
- Peak hours: 8–11 AM | Top days: Monday & Tuesday
- 95.8% card payment penetration

## Tools Used
Microsoft Excel — Pivot Tables, Data Analysis, Reporting

- `Coffee_data.xlsx` — Raw dataset
